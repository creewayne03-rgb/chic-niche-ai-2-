## Quick Start (Windows)
1. Right-click `START_WINDOWS.bat` → Run as administrator
2. Open browser → `http://127.0.0.1:9000/app`

## Deploy Online
1. Push to GitHub
2. Connect to railway.app
3. Add `GROQ_API_KEY` environment variable
4. Share your public URL!
